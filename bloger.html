1. What is DOM and what is the purpose of DOM?

Ans: Dom means Document Object Model (DOM) and it is make a tree-like representation nodes and objects of a webpage's content .The purpose of Dom create dynamic and interactive web page can to respond and to user interaction .

2. How will you select HTML elements using DOM? Name the DOM methods?

Ans: Generally the five way to select elements in a DOM using selectors.
example :
1. getElementsByClassName() method returns a collection of elements with a specified class name .
2. getElementById() this method use to returns an element with a specified value .
3. getElementsByTagName() this method use to collection of all elements with a specified tag name .
4. querySelector() this method returns the first element that matches a CSS selector.
5. querySelectorAll() this method returns all elements that match a CSS selector .

4. What is a callback function and why will you use it?

Ans: A callback function is one that is passed as an argument to another function and then executed within the parent function.Callbacks are frequently used in asynchronous programming, where a function may need to wait for an operation to complete before proceeding. The callback function is used in this case to specify what should happen after the operation is finished.
